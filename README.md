# Matter Of Size
## Story / Concept
Myriad of escape rooms, with the power of shrinking and growing along with items held, solve puzzles set in a modern style architecture, with whimsical mysteries and out of the box answers, will you solve all escape rooms' puzzles and make it to the end?

## GameJam
Entry for Game Maker's Toolkit GameJam 2024
